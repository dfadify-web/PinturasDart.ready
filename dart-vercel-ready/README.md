# DART Painting - Website

Sitio web profesional para DART Painting, especializado en transformación de espacios residenciales y empapelado premium.

## Tecnologías
- HTML5 / CSS3
- JavaScript (GSAP, ScrollTrigger, Lenis)
- Despliegue en Vercel

## Cómo desplegar
1. Sube estos archivos a un repositorio de GitHub.
2. Conecta el repositorio en [Vercel](https://vercel.com).
3. Vercel detectará automáticamente que es un sitio estático y lo desplegará.
